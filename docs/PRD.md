# 개인용 학습노트 MVP PRD

## 🎯 핵심 정보

**목적**: Notion을 CMS로 활용해, Notion에서 글을 작성하면 별도 배포 작업 없이 웹사이트에 자동으로 반영되는 개인 학습 노트를 제공한다.
**사용자**: Notion에 학습 내용을 기록하고 로컬 웹사이트에서 다시 찾아보는 1인 사용자(본인).

## 🚶 사용자 여정

```
1. 홈 (최근 글 목록)
   ↓ 진입 시 Notion DB에서 글 목록 자동 로드

2. 목록 탐색
   [검색어 입력]      → 검색 결과로 목록 필터링 → 다음 단계
   [카테고리 클릭]     → 태그 기준으로 목록 필터링 → 다음 단계
   [필터 없음]         → 전체 최근 글 목록 → 다음 단계
   ↓

3. 글 카드 클릭
   ↓

4. 글 상세 페이지 → 본문/태그/작성일 확인
   ↓ 뒤로가기

5. 홈으로 복귀 → 다른 글 탐색 반복
```

## ⚡ 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F001** | 글 목록 조회 | Notion 데이터베이스에서 글 목록(제목, 태그, 작성일)을 가져와 카드 형태로 표시 | Notion을 CMS로 쓰는 핵심 가치(자동 반영)의 진입점 | 홈 |
| **F002** | 글 상세 조회 | 선택한 글의 Notion 페이지 본문(Content)을 렌더링 | 학습 노트 열람이라는 핵심 목적 그 자체 | 글 상세 |
| **F003** | 카테고리(태그) 필터링 | Tags 속성을 기준으로 목록을 필터링 | 기록이 쌓일수록 원하는 주제를 빠르게 찾기 위한 필수 탐색 수단 | 홈 |
| **F004** | 검색 기능 | 제목/본문 기준으로 글을 검색 | 태그만으로 못 찾는 글을 찾기 위한 필수 탐색 수단 | 홈 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|-------------|------------|
| **F005** | 반응형 레이아웃 | 모바일/태블릿/데스크톱 화면 크기에 맞춰 목록·상세 뷰가 자연스럽게 재배치 | 로컬 PC 외 모바일 브라우저로도 노트를 확인하기 위한 최소 요건 | 홈, 글 상세 |

### 3. MVP 이후 기능 (제외)

- 다중 사용자/로그인 (개인 전용 로컬 사용이므로 인증 불필요)
- 글 작성/수정 UI (Notion 자체에서 작성, 웹사이트는 조회 전용)
- 댓글, 좋아요 등 소셜 기능
- 다크모드 외 테마 커스터마이징, 알림 설정
- 고급 검색(정규식, 태그 조합 검색 등)

## 📄 페이지별 상세 기능

### 홈

> **구현 기능:** `F001`, `F003`, `F004`, `F005` | **메뉴 위치:** 최상단 진입 페이지

| 항목 | 내용 |
|------|------|
| **역할** | 전체 학습 노트를 최신순으로 훑어보고 원하는 글을 찾는 진입 화면 |
| **진입 경로** | 사이트 접속 시 기본 진입 |
| **사용자 행동** | 최근 글 목록을 스크롤하며 훑어보거나, 검색어를 입력하거나, 카테고리(태그)를 클릭해 목록을 좁힌다 |
| **주요 기능** | • Notion DB 기반 최근 글 목록 카드 표시(제목/태그/작성일)<br>• 검색어 입력 시 실시간 목록 필터링<br>• 태그 클릭 시 해당 카테고리 글만 필터링<br>• 화면 크기에 따라 카드 그리드 열 수 자동 조정 |
| **다음 이동** | 글 카드 클릭 → 글 상세 |

---

### 글 상세

> **구현 기능:** `F002`, `F005` | **메뉴 위치:** 홈에서만 진입 가능

| 항목 | 내용 |
|------|------|
| **역할** | 선택한 학습 노트 한 편의 전체 내용을 읽는 화면 |
| **진입 경로** | 홈에서 글 카드 클릭 |
| **사용자 행동** | 본문을 읽고, 태그·작성일을 확인하고, 다 읽으면 목록으로 돌아간다 |
| **주요 기능** | • Notion 페이지 본문(Content) 렌더링<br>• 제목/태그/작성일 표시<br>• 화면 크기에 따라 본문 폭·여백 자동 조정 |
| **다음 이동** | 뒤로가기 → 홈 |

## 🗄️ 데이터 모델

### Post (Notion 데이터베이스 항목)

| 필드 | 설명 | Notion 속성 타입 |
|------|------|----------|
| Title | 글 제목 | title |
| Tags | 카테고리 분류용 태그(다중 선택) | multi_select |
| Content | 본문 | page content(blocks) |
| CreatedAt | 작성일 | date |

## 🛠️ 기술 스택

### 🎨 프론트엔드 프레임워크

- **Next.js** (App Router) — React 풀스택 프레임워크
- **TypeScript** — 타입 안전성 보장

- **Notion API** (`@notionhq/client`) — Notion 데이터베이스 조회 및 페이지 본문 렌더링

- **Tailwind CSS** — 유틸리티 CSS 프레임워크
- **shadcn/ui** — React 컴포넌트 라이브러리
- **Lucide React** — 아이콘 라이브러리

### 🚀 배포

- **localhost** — 로컬 개발 서버로만 운영, 별도 호스팅 없음
